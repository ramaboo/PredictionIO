# PredictionIO Documentation

## Starting the Server

Start the server with:

```
$ bundle exec middleman server
```

This will start the local web server at [http://localhost:4567](http://localhost:4567/).

## Building the Site

Build the site with:

```
$ bundle exec middleman build
```
